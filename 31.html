    • gromacs5.0+的详细安装

    2015-12-02
    • Gromacs

    gromacs 5以上安装版本的流程与方法是一样的，乃至于4.6X版本同样可以使用这种方法，我这里以5.0.2版本为例进行的安装。

    默认安装在各位文件夹（我的用户名kangsgo）的mdinstall下：

    一.预先安装：

    sudo apt-get install gcc

    sudo apt-get install g++

    sudo apt-get install gfortran

    PS：若你只需要安装4.6.X单精度并行版，只需要：

    sudo apt-get install gromacs

    后面无需在看

    二.安装cmake

    方案一：直接apt-get

    sudo apt-get install cmake

    方案二：本地安装

    1.进入文件目录：

    cd /home/kangsgo/mdinstall

    2.解压缩：

    tar -xvf cmake-3.3.2.tar.gz

    3.进入目录：

    cd cmake-3.3.2

    4.设置，附加安装路径，我将其安装在cmake文件夹下

    ./bootstrap –prefix=/home/kangsgo/mdinstall/cmake

    5.安装

    make

    make install

    6.设置路径

    gedit ~/.zshrc (1.ubuntu默认为bashrc命令，我这里改为了zshrc，下同，所以你可能下面所有的zshrc都需要改为bashrc 2.倘若你安装在/usr/目录下的话需要sudo获得权限，个人建议最好不要sudo su或者最高权限后进行操作，因为那样子所有的文件都是高权限的)

    最底部增加

    export PATH=”$PATH:/home/kangsgo/mdinstall/cmake/bin/:.”

    其中home/kangsgo/mdinstall/cmake/需要替换为你的路径，下同

    source ~/.zshrc

    6*.查看是否安装好

    cmake

    三.安装fftw

    方案一.gromacs安装时集成[推荐]

    方案二.自行编译安装

    tar xvf fftw-3.3.4.tar.gz

    cd fftw-3.3.4

    ./configure –enable-DFFTWF_INCLUDE_DIR –enable-shared

    make

    sudo make install

    设置路径，或者gromacs那一步设置，由于不推荐这种方法，就不过多累述

    四.安装openmpi

    1.同样先要进入开始放置下载的目录：

    cd /home/kangsgo/mdinstall

    2.解压缩和进入目录,将其安装在openmpi目录下：

    tar xvf openmpi-1.10.tar.gz

    cd openmpi-1.10

    ./configure –prefix=/home/kangsgo/mdinstall/openmpi

    3.安装：

    make all

    make install

    4.路径设置：

    gedit ~/.zshrc

    openmpi

    export LD_LIBRARY_PATH=/home/kangsgo/mdinstall/openmpi/lib:$LD_LIBRARY_PATH

    export PATH=/home/kangsgo/mdinstall/openmpi/bin:$PATH

    source ~/.zshrc

    五.CUDA驱动安装

    由于amber和gromacs等许多软件都需要用的，我另起一篇文章，具体可以点我查看

    注意两种方法都需要设置路径：

    CUDA

    export PATH=”$PATH:/usr/local/cuda-7.5/bin”

    export LD_LIBRARY_PATH=”$LD_LIBRARY_PATH:/usr/local/cuda-7.5/lib64“

    六.GROMACS的安装

    千呼万唤始出来，终于到了gromacs的安装了～其实gromacs的安装反而更加简单明了，我把其安装在/home/kangsgo/mdinstall/md/下面

    1.进入目录解压进入目录（由于是给予新手帮助，原谅我啰嗦）

    cd /home/kangsgo/mdinstall/md/

    tar xfz gromacs-5.0.2.tar.gz

    cd gromacs-5.0.2

    2.创建文件夹：

    mkdir build

    3.进入文件夹

    cd build

    4.编译

    cmake .. -DGMX_BUILD_OWN_FFTW=ON -DGMX_GPU=ON -DGMX_MPI=ON -DCMAKE_INSTALL_PREFIX=/home/kangsgo/mdinstall/md/gromacs
    倘若FFTW不是自动安装，要手动安装的话需要更改为：
    -DGMX_FFT_LIBRARY=fftw3 -DFFTWF_LIBRARY=”/usr/local/lib/libfftw3f.so” -DFFTWF_INCLUDE_DIR=”/usr/local/include/“
    这里就不详细说明了

    5.安装

    make -j 8（8表示make时使用8核，如果只用一个核，只需使用make）

    make check

    make install

    6.设置路径

    gedit ~/.zshrc

    gromacs

    source /home/kangsgo/mdinstall/md/gromacs/bin/GMXRC

    source ~/.zshrc
    到此gromacs安装到此结束！
